To-Do List Web Application

Project Overview

The To-Do List Web Application is a lightweight task management system designed to help users organize daily activities efficiently. The application allows users to create, update, complete, and delete tasks through a simple and user-friendly interface. The same web application is packaged as an Android app for mobile usage.

Features

⦁	Add new tasks

⦁	Edit existing tasks

⦁	Mark tasks as completed

⦁	Delete tasks

⦁	Responsive and simple user interface

⦁	Works without server or database

⦁	Android APK support


Technologies Used

⦁	HTML

⦁	CSS

⦁	JavaScript

⦁	WebIntoApp (for Android conversion)


How to Run the Web Application

⦁	Open the code folder.

⦁	Double-click index.html.

⦁	The application will open in your default web browser.

⦁	No installation, server, or database setup is required.

⦁	Android Application:

   -app-release.apk can be installed directly on an Android device.

   -app-release.aab is used for Play Store deployment.

⦁	The application is converted from web to Android using WebIntoApp.

Algorithm Description 
1.	The application accepts user task input and stores it in memory using JavaScript.

2.	Tasks are dynamically rendered on the user interface using DOM manipulation.

3.	User actions such as add, update, complete, and delete trigger real-time UI updates.

System Requirements

1.	Web Browser (Chrome, Edge, Firefox)

2.	Android device (optional, for APK testing)
